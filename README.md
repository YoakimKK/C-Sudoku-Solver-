# C-Sudoku-Solver-
Sudoku C solver using backtracking and dynamic memory allocation to solve puzlles of different sizes(e.g, 9x9, 16x16, etc.) Program handles the dynamic creation of the Sudoku grid and the backtracking algorithm to solve the puzzle.

This was made for a school project in C language class, it's sole use is to quickly solve sudoku puzzles of various sizes. As the user you have to implement the sudoku size as well as the the rows. For rows accepts ints seperated by space, each row has to be the number equal to the size of grid number.
